Inspection Process

  1. Request Inspection: Submit request at least 24 hours in advance
  2. Confirmation: Receive confirmation of inspection date
  3. Prepare Site: Ensure work area is accessible and ready
  4. Inspector Arrival: Inspector visits between 8:00 AM - 4:00 PM
  5. Inspection: Inspector examines work and documents findings
  6. Results: Pass/fail notice posted on-site and emailed
  7. Corrections: If failed, make corrections and request re-inspection

Inspection Results

Check Inspection Status

Result Types

  • Approved: Work meets code requirements, proceed to next phase
  • Approved with Conditions: Minor items noted, may proceed with corrections
  • Failed: Corrections required before re-inspection
  • Canceled: Inspection canceled (not ready, access issues, etc.)

Re-Inspections

If inspection fails:

  • Make all required corrections
  • Request re-inspection (no additional fee for first re-inspection)
  • Additional re-inspections: $75 fee

Frequently Asked Questions

When should I call for an inspection?

Call when work is complete and ready for inspection, but before it's covered up. For example, call for framing inspection before installing drywall.

Do I need to be present for the inspection?

Not required, but recommended. Ensure the inspector has access to the work area.

What if I fail an inspection?

The inspector will note required corrections. Make corrections and request a re-inspection.

How long is an inspection good for?

Approved inspections remain valid for the life of the permit (typically 6-12 months).
