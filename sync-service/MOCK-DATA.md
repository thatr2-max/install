## Quick Setup Instructions

### Step 1: Create the Spreadsheets
1. Go to Google Sheets (sheets.google.com)
2. Create 7 new spreadsheets with these exact names:
   - Municipality Config
   - Staff Directory
   - Events
   - Public Notices
   - Job Postings
   - Boards & Commissions
   - News

### Step 2: Paste the Data
1. For each spreadsheet, copy the entire data block above (including the header row)
2. Click on cell **A1** in the empty spreadsheet
3. Paste (Ctrl+V or Cmd+V)
4. The tab-separated data will automatically fill the columns

### Step 3: Verify the Layout
- Each spreadsheet should have data starting in row 1 (headers)
- Data should be spread across multiple columns
- No extra columns or rows should be present

### Step 4: Share with Service Account
1. Open each spreadsheet
2. Click "Share" button
3. Add your service account email (something like `xxxx@yyyy.iam.gserviceaccount.com`)
4. Give "Editor" permissions
5. Click "Send"

### Step 5: Get Sheet IDs
For each spreadsheet:
1. Look at the URL: `https://docs.google.com/spreadsheets/d/SHEET_ID_HERE/edit`
2. Copy the `SHEET_ID_HERE` part
3. Save it for your config file

### Step 6: Test the Sync
```bash
cd /home/user/install/sync-service
python simple_sync.py --config simple_sync_config.json --once
```
